## Self-Healing Events
- Step 3: First `patchValue({timeInForce:'GTC'})` did not render expiration field. Healed by dispatching a `change` event on the underlying `<select>` element after patchValue, which triggered Angular to re-evaluate the conditional `*ngIf` and render Month/Day/Year inputs. Cause: ReactiveForms patchValue updates model but the template's conditional render keyed off the native select event listener. Action: dispatch change event. Result: OK.

## Lessons Learned During Run
- For Angular reactive forms with conditional fields keyed off select changes (e.g., expirationDate gated by timeInForce=GTC), patchValue alone may not trigger conditional template rendering. Dispatch a manual `change` event on the underlying `<select>` after patchValue.

## Strategy Changes
- After patchValue on a select that controls a `*ngIf`, always dispatch a change event on the corresponding native select before reading dependent state.
