## Lessons Learned During Run
- The Angular reactive form enforces `required` on `expirationDate` only when `timeInForce === 'GTC'`. Observed expErr={required:true} when GTC set with no expiration.
- The DOM `<input type="date">` for expiration did not appear in the current snapshot, but the form-level validator caught the missing field, so the behavioral contract held. (UI polish finding, not a functional failure.)

## Strategy Changes
- For conditional reactive-form fields, assert via `form.get(...).errors` and `form.valid` rather than relying on DOM rendering of the input.
