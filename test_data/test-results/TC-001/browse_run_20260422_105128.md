## Self-Healing Events
- Step 6: Queue view on /queue after auto-redirect was stuck with loading=true and trades=0. Reloaded via `agent-browser open /queue` which completed the data fetch. Result: OK.

## Lessons Learned During Run
- After onSubmit() auto-navigates to /queue, issue a fresh `agent-browser open /queue` to force the queue service fetch to resolve.
- The Angular app has ~1.5-2s simulated latency; always wait 2-4s after navigation before reading state.

## Strategy Changes
- Use `agent-browser eval` with `window.ng.getComponent(...)` for form patching and component-state introspection instead of DOM selection.
