## Self-Healing Events
- Step 3: `patchValue` on marketSector didn't trigger the ticker dropdown DOM re-render in Angular even though the component's `tickerOptions` array updated correctly. Fix: dispatch a native `change` event on the <select> so Angular's valueChanges pipeline runs and the view re-renders.

## Lessons Learned During Run
- For cascading Angular reactive-form selects, `patchValue` updates the model but may not render dependent <option> lists. Dispatch a native `change` event on the source <select> to force the dependent re-render.
- Floating-point totals like 836.70 will appear as 836.6999… when read from the form — compare with a tolerance or `.toFixed(2)`.
